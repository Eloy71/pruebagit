Ver en que rama nos encontramos: git branch

crear una nueva rama: git branch 'nombre de la rama a crear'

editar el nommbre de la rama: git branch -m rama1'nombre de la rama a cambiar el nombre' ramitata'nuevo nombre de la rama'

para pasar a otro rama: git checkout 'nombre de la rama a pasar'

para elimiar una rama: git branch -d 'nombre de la rama a eliminar'

solo se puede elimar una rama si no te encuentras en ella
